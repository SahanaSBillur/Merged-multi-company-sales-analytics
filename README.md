# Merged-multi-company-sales-analytics
1 Project Name and Headline : Multi-Company Sales Performance Analysis (2019–2025)
Headline: Built an Excel-based analytics dashboard to evaluate year-wise, region-wise, and product-wise sales, revenue, and profit trends using Pivot Tables and KPIs.

2 Key Analysis Performed :
Created dynamic Pivot Tables to summarize sales by product and category
Data is distributed across Years (2019–2025), Regions, and Categories
Most entries are concentrated in North, South, and West regions

3 Year-wise Insights :
2020–2024 each have 4 records → consistent activity
2019 has only 2 records → lowest
2025 also drops to 2 records
There is no strong growth trend, but a stable middle period (2020–2024)

4 Insight:
Prices are highly scattered
No dominant price point
Likely sample data, not enough for pricing trend analysis
